## Why

Разработка инклюзивного Android-приложения "FigaTram" для отслеживания трамваев на линии T3 (Анталья) в реальном времени. Приложение позволит не только планировать поездки маломобильным гражданам (выделяя низкопольные трамваи № 152 и № 175 и рассчитывая для них ETA), но и собирать базу данных трамваев ("справочник споттера"), позволяя просматривать информацию о каждом трамвае по клику на карте в стиле Flightradar.

## What Changes

- Создание нового Android-проекта с использованием Kotlin и Jetpack Compose
- Интеграция Google Maps SDK для отображения карты, маршрута линии T3 (всех остановок и самой линии), а также всех трамваев на линии
- Подключение к API для поллинга координат трамваев каждые 20 секунд
- Реализация функционала "карточка трамвая" (просмотр подробной информации и фото при клике на движущийся маркер)
- Ведение локального справочника трамваев с возможностью отмечать "любимые"
- Реализация фонового мониторинга и расчет ETA для целевых (инклюзивных и любимых) трамваев
- Отправка локальных пуш-уведомлений за настраиваемое количество минут до прибытия на выбранные остановки
- Сохранение статистики прибытия целевых трамваев на станции

## Capabilities

### New Capabilities
- `tram-tracking-api`: Поллинг API с интервалом 20 секунд, извлечение координат для **всех** трамваев.
- `map-ui`: Отображение карты Google Maps, линии маршрута T3, маркеров остановок с названиями, а также стандартных маркеров для обычных трамваев и специализированных (инклюзивных) для 152 и 175. Поддержка клика по маркеру для открытия карточки.
- `tram-directory`: Локальная база данных (Room) и экраны для справочника трамваев (добавление фото, характеристик, отметка "любимый").
- `eta-notifications`: Выбор нескольких остановок и гибкого времени оповещения. Фоновый сервис для отслеживания расстояния и отправки Push-уведомлений.
- `arrival-statistics`: Сбор, хранение и отображение статистики прибытия целевых трамваев.

## Impact

- **Новый проект**: Инициализация с нуля. Базовая архитектура (MVVM, Clean Architecture).
- **Сетевой слой**: Интеграция с внешним сервисом отслеживания транспорта, настройка SecretConfig.
- **UI**: Разработка пользовательского интерфейса (Карта, Справочник, Карточка трамвая, Настройки) с использованием Jetpack Compose и Material 3.
- **Хранение данных**: Локальная база данных Room для справочника, аналитики и истории.
