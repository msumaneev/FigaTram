## ADDED Requirements

### Requirement: Отображение карты и всех трамваев
Система MUST отображать Google Maps на главном экране и размещать маркеры для **всех** активных трамваев, полученных из API.

#### Scenario: Отрисовка всех трамваев
- **WHEN** получены координаты трамваев
- **THEN** на карте отображаются маркеры для каждого трамвая

### Requirement: Визуальное выделение инклюзивных трамваев
Система MUST визуально выделять целевые трамваи (152, 175) с использованием специальных маркеров (иконка инвалидной коляски или зеленый цвет).

#### Scenario: Подсветка доступных трамваев
- **WHEN** маркер принадлежит трамваю с ID "152" или "175"
- **THEN** для него применяется инклюзивный дизайн маркера

### Requirement: Карточка трамвая по клику (Flightradar-style)
Система MUST обрабатывать клики по маркерам трамваев и отображать всплывающую карточку (BottomSheet) с подробной информацией о трамвае.

#### Scenario: Открытие карточки
- **WHEN** пользователь нажимает на маркер движущегося трамвая
- **THEN** открывается карточка трамвая с его бортовым номером, фото (если есть) и другими характеристиками

### Requirement: Плавное перемещение маркеров
Система MUST плавно анимировать перемещение маркеров между старыми и новыми координатами.

#### Scenario: Обновление координат
- **WHEN** API возвращает новые координаты для трамвая
- **THEN** маркер плавно перемещается из текущей позиции в новую

### Requirement: Отрисовка маршрута и остановок
Система MUST отображать на карте линию маршрута T3 (Polyline) и все остановки маршрута (маркеры с названиями станций).

#### Scenario: Отображение инфраструктуры линии T3
- **WHEN** карта успешно загружена
- **THEN** на ней рисуется линия маршрута T3 и расставляются маркеры остановок с подписанными названиями
