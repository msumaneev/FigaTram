## ADDED Requirements

### Requirement: Поллинг внешнего API
Система MUST выполнять HTTP-запросы (с отключенным SSL Pinning и секретными заголовками из SecretConfig) к внешнему API отслеживания с частотой 20 секунд для получения актуальных координат транспорта.

#### Scenario: Успешное получение данных
- **WHEN** срабатывает таймер поллинга (каждые 20 сек)
- **THEN** система выполняет запрос и успешно парсит ответ, содержащий массив vehicles

### Requirement: Извлечение данных обо всех трамваях
Система MUST извлекать из ответа координаты и данные **всех** трамваев на линии, обновляя их текущие координаты (`position.latitude`, `position.longitude`) и `stop_id`.

#### Scenario: Обновление состояния трамваев
- **WHEN** получен ответ API со списком всех трамваев
- **THEN** система обновляет в локальном состоянии (Flow) координаты всех трамваев для их последующей отрисовки
